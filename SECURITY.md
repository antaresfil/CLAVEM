# Security policy (responsible disclosure)

## Reporting a vulnerability

Please report security issues **privately**.

- Email: **clavemhelp@noxfarm.com**
- Subject: `CLAVEM Security Report`

Include:
- affected version / commit
- clear reproduction steps or PoC
- impact assessment (what an attacker gains)
- any suggested fixes (optional)

## What to expect

- You will receive an acknowledgement as soon as possible.
- If confirmed, a fix will be prepared and released, and you will be credited if you want (or kept anonymous).

## Please do not

- open public issues for vulnerabilities before a fix is available
- publish exploit code before coordinated disclosure

Thank you for helping improve CLAVEM.
